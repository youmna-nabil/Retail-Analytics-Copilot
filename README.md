# Retail Analytics Copilot (DSPy + LangGraph)

## Overview
This repository is dedicated to implementing **building a local AI agent** Uses Northwind SQLite DB. This database sample was provided with Microsoft Access as a tutorial schema for managing small business customers, orders, inventory, purchasing, suppliers, shipping, and employees.


## Prerequisites
To run the code in this repository, ensure you have the following installed:
- Python 3.11 or higher
- dspy-ai
- LangGraph


Install dependencies using:
```bash
pip install -r requirements.txt
